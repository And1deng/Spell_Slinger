#Spell Slinger

A 2D top down survival game build in Lua and the LOVE2D framework. This project was built for personal learning and to explore game architecture such as state machines, object management, enemy AI, and world generation.

The player is able to cast spells using arrow key sequences and defeat enemies. Along with that, players have dodge movement to avoid enemy attacks. The project also implements an extensible game architecture which allows new spells, enemies, and enemy AI behaviors to be easily added. 

#Features
User input via WASD and arrow keys
Different spell projectiles to cast with varying stats
Two enemy types for varying gameplay
Modifiable enemy, world, and player numbers
Basic UI

#Requirements 
Windows
LOVE2D
Visual Studio Code

#Build Instructions
Open the file in the terminal and run "love ."

#Controls
WASD movement
UP, DOWN, LEFT, RIGHT for spell casting
As of now, there are only three spells
FIREBALL - UP UP UP
ICE SHARD - RIGHT RIGHT RIGHT
BOULDER - DOWN DOWN DOWN
